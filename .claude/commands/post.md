# /post — Post Video to YouTube

> **Not yet implemented.**

This command will help you upload and schedule videos on YouTube using the YouTube
Data API v3.

To add this feature, create:
- `directives/publishing/post.md`
- `orchestrators/publishing/publish-pipeline.md`
- `executors/publishing/upload_video.py`

Note: this will require a Google OAuth 2.0 client credentials file.
